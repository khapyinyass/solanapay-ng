# SolanaPay NG

A simple Solana Pay web app for Nigerian vendors to accept SOL/USDC via QR code.

## Features
- Connect Phantom Wallet
- Generate Solana Pay QR code
- Track payments (demo)

## How to Run
1. Open `index.html` in browser
2. Click "Connect Wallet"
3. Click "Generate Payment QR"

## Status
MVP ready, open-source, under active development
